# 🛒 E-commerce Sales Analysis

📊 Project Overview

This project analyzes retail sales data to identify trends, top-performing products, and business insights.

🛠 Tools Used

* Python (pandas, matplotlib)
* Power BI

📈 Key Analysis

* Sales and profit analysis by category
* Monthly sales trend
* Region-wise performance
* Loss-making products identification

Key Insights:

* Technology category generated highest revenue
* Some products resulted in consistent losses
* Sales increased during year-end months
* Discounts negatively impacted profit
